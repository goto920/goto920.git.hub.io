1. JavaScript metronome demo

The progmam starts immediately after you click the https://goto920.github.io/demos/metronome/.
Source code: https://github.com/goto920/nodejs/metronome/.

Screen shot Description

Basic Operation -- Select Beat, Adjust BPM, then press Start

  1. Select Beat: 4/4, 6/8, 8/8 swing etc.
  2. (My) voice(default off): My voice of 1, 2, 3, 4 in addition to cowbels
  3. Adjust bpm(0.1 step) with the range slider (play continues)
    or enter/modify number (play stops)
  4. Start/Stop
  5. Timer(sec): Similar to kitchen timer. Set positive number for expiration time in second. The timer pauses when the metronome is stopped. Set 0 to disable the timer.

Advanced Options

  1. Swing adjust(0.0 to 3.0): 1.5(straight), 2.0(full swing)
  2. Increment: automatic bpm step up.
    Set bpm and bars (set 0 to stop)
  3. Mute: mute bar(s) at the probability set.
    Note: Always sound is turned on after the mute bar length.
    For example, 1 bar(s) at prob. 1.0 gives alternate on/off.
