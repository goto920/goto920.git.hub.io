Brief How To

Prepare audio file (wav, mp3, and other formats depends on the device). The folders, which iPhone mobile Safari can access, are quite limited. iCloud Drive might be a good choice.
  • 1) Load an audio file in the top input box.
  • 2) "PlayFromA" to play once (A is default 0), "Pause" to pause
  • "LoopAB" to play AB loop, "StopLoop" to stop loop
  • Speed Control: move the slider (25% to 200%)
  • Pitch Control: move the semi-tone slider and fine pitch adjust ment by moving "cents" slider (100 cents = 1 semi-tone) LI>(**Only if necessarary**) 3) "ExportWav" to export wave file at the selected speed and pitch
  • Notes on ExportWav
    • ExportWav needs playback time with the current processing (slow playback needs more time)
    • Volume Control does not affect the volume of exported wav file.
    • Do not move Speed and Pitch Sliders while exporting
    • Filename will be set like "original&s90&p-1.wav" with speed and pitch value.
iPhone requires special operation to store the exported data. Blob window pops up and it is possible to save the data in iCloud Drive. You want to rename the file with extension".wav", because the filename will not be set automatically. I know it is stupid to wait for playback time for export.
