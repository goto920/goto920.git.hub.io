概要

  1. インストール操作不要: JavaScriptなので当たり前ですが、古くない Android, iOS, Windows, macOS, Linux等のWebブラウザで動くはずです。
    (動作検証機器: Android(SO02H)、iOS(iPod 中古買いました)、 Windows(Surface Pro3)、macOS(mac mini、サーバ用なのでたまに)
  2. メニュー言語: 上の「JP」で日本語に切り替え。 Webブラウザの最優先言語で自動設定したいが、Webブラウザ依存で、 色々な機器での確認が面倒なので、まずは手動切り替え。
  3. ネットワーク接続: ファイルサイズは1MB未満。 一度アクセスして、ブラウザにキャッシュされたら電波が届かなくてもOK (スマートフォンのキャッシュは強烈なのでまず大丈夫)
  4. タイムの正確さ: 実験1回だけですが、極端に速いテンポでなければ、 大丈夫そうです。200BPM超はせいぜい8分音符にしてください。
    -- 60秒のはずが61秒などは、機器の内部クロックに依存する現象なので、 がまんしてください。
  5. 報告された問題点(主に基本機能)
    • オフライン利用-- PCやAndroidでは可能だがiPhone(iOS)では不可。 2017年6月現在の情報に基づくと、iOSではWebブラウザをChrome やFirefoxに変えてもオフライン利用はできないようです。 ただし、ネットワーク接続してメトロノームを起動した後は、ネットワークが切れても利用できるはずです。
    • MicroSoft Edge(Windows 10)で調子悪い(4/4の速いのは出ても、 8/8の音が出ない) -- 修正完了(1217am)
    • iOS11.2(iPhone7)で"Start/Stop" ボタンがスタート後反応しないのでStopできない(20171212) -- Webブラウザ閉じる。 (いくつか対策してみた)
    • 使っているうちに音が出なくなる -- Webブラウザの*すべての*タブや Windowを終了(いくつか対策してみた)
    • 動かない -- JavaScriptは有効ですか?
    • 音が出ない-- イヤフォンから音が出れば正常。 スピーカーの音量設定を確認

基本操作 -- ビート、BPM を設定してStartを押す

  1. ビート選択: 4/4, 6/8, 8/8swing(シャッフル) などをプリセットのリストから選ぶ
  2. 音選択: (カウ)ベル/声/両方
  3. Start/Stop
  4. BPM(4分音符)はスライダでは1BPM単位、さらにセレクタで0.1単位で設定可能
    「タップ」でも設定可能(3秒でリセットされる)
  5. (減算)タイマ: 秒、または小節で指定
    Stopするとタイマは一次停止、Startで続き、 0にするとタイマ無効にできる。

発展(ドラマー向き)

  1. スイング調整 
    3連音を0,1,2,0,1,2 として説明を読んでください。

前の音                     次の音
0            1.5   2.0      3.0   
0----x-----------------------0 (0.5以上1.5未満: スイングの逆)
0-------------x--------------0 ☆ (1.5: ストレート)
0----------------x-----------0 (1.8: 軽いスイング) 
  16ビートのシャッフルでよくあるちょいハネ練習を想定
0------------------x---------0 ☆ (2.0: (ふつうの)スイング)
0--------------------x-------0 (2.2: 重いスイング) 
0-----------------------x----0 (2.5: 極端なスイング)
  2. (テンポ)自動up/down: -10から10bpmまでを、 1, 2, 4, 8, 12, 16小節単位で自動的にup/down変更
    一旦Stopして、bpmまたは小節を0にすると無効にできる
  3. ランダムミュート: ミュートする長さを小節で指定(0で無効)、確率を0から1の間で設定する。
    注: 休み小節の次は必ず復帰して音が出る。 例えば、1小節休みで確率1の場合、音あり、なしの小節が交互になる。
  4. 偶数音Vol (頭の音を小さく、3連の場合は2つめまで小さくする) 
    O: フルボリューム
x: 消去(0.0) or 小さく(0.1))
----------------------
4/4 : x     O   x   O (使わないでしょう)
6/8 : x O x O x O     (Wartz style)
8/8 : x O x O x O x O (裏打ちや裏メトロノームでの練習)
12/8: x x O x x O x x O x x O (3連の最後の音に合わせて練習
----------------------
注: このメトロノームでは 12/8 = 8分音符3つ x 4 times が1小節
  5. カスタムループ: 「表示」を押すとループ編集画面が追加される (機能確認中)。

スクリーンショット 説明
  • カスタムループのボタン: 「表示」/「隠す」の切り替え
  • 表の操作 (表が空だとStartできない)
    • 最後の行でビート(プリセット)と回数を選んで、aの横のボタンで 最下行に追加(挿入はできない)
      -- swingは、プリセットビートの値から変更したい場合だけ編集する
    • 登録済の行はdの横のボタンで消去できる
    • 登録された行の再生が終わると先頭に戻ってループ再生する
  • Startボタン: ループ再生開始/終了
  • 「巻き戻し」ボタン: ループの先頭に戻る。再生中の場合は、再生停止して戻る
